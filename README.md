# Intro
Intro(на базе StructSample) - это плагин для Autodesk Revit, написанный на C#. Он автоматизирует создание и расстановку несущих колонн вдоль выбранных пользователем стен с заданным интервалом.
## Краткое описание
Основная логика включает:
- Поиск в проекте определенного типа колонны (семейства) по имени.
- Вычисление точек для расстановки колонн вдоль стены с фиксированным интервалом.
- Создание экземпляров семейства колонн в этих точках.
- Настройка параметров колонн, таких как базовый и верхний уровень.
> Важное примечание: Тип колонны и интервал между ними жестко заданы в коде (хардкод).
## Инструкция по использованию
- В открытом проекте выберите одну или несколько стен.
- Запустите внешнюю команду StructSample.
- После выполнения скрипта вдоль каждой выбранной стены с интервалом 5 футов будут расставлены колонны.
## Ключевые классы и методы Revit API
В проекте используются следующие основные классы и методы Revit API.
### Интерфейс: Autodesk.Revit.UI.IExternalCommand
### Основные классы:
- Autodesk.Revit.DB.Wall
- Autodesk.Revit.DB.Family
- Autodesk.Revit.DB.FamilySymbol
- Autodesk.Revit.DB.FamilyInstance
- Autodesk.Revit.DB.Parameter
### Геометрия:
- Autodesk.Revit.DB.XYZ
- Autodesk.Revit.DB.UV
- Autodesk.Revit.DB.Line
### Ключевые методы:
- Document.Create.NewFamilyInstance - для создания новой колонны.
- Family.Symbols - для получения всех типов в семействе.
- FamilyInstance.get_Parameter() - для работы с параметрами экземпляра (например, BuiltInParameter.FAMILY_BASE_LEVEL_PARAM).
